💡 使用方法

  • Magic Link認証: 上記メールアドレスでMagic Linkをリクエスト
  • テストユーザーログイン: ユーザー番号（1-5）を選択してログイン
  • ID/パスワード認証: 上記ユーザーID・パスワードを入力して「ユーザー情報認証テスト」を実行
  • 開発用ログイン: 任意のメールアドレスで開発者/管理者モードでログイン
⚠️ 重要:
  • これらのサンプルユーザーは開発環境専用です
  • 本番環境では使用できません
  • パスワードは平文で管理されているため、実際のシステムでは使用しないでください
